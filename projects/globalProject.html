Project Description

The calendar displays the current month, current days weather based on users computer location, current days task along with any holidays for current day. The calendar will automatically switch the months, the user doesn't have the ability to change the month. The user has the ability to add/delete/update task status for any day of the month, which is stored in the local storage. The current weather is displayed for the users login location along with user location/date/humidity/windspeed for their current day. The user has the ability to select different countries that will automatically display that countries holidays in the calendar, and current news articles for that country.
